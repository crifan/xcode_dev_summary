# Xcode开发心得

* 最新版本：`v0.7.1`

## 简介

总结Xcode相关的开发心得。先是概览；然后总结Xcode开发的心得，包括安装和升级Xcode；Command Line Tools命令行工具，包括xcodebuild和xcrun；以及Signing签名、iOS设备的iPhone和iOS模拟器；日志，包括调试和崩溃；以及其他心得；以及相关子教程。
